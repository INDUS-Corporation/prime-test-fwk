A. Eclipse setup instructions	
	- Maven should be configured in eclipse

B. Code compilation and test suite execution using maven
	- use the following command to clean the build
		> mvn clean
	- use the following command to create the jar file
		> mvn clean package
	- use the following command to install the jar file in maven repository.
		> mvn clean install
